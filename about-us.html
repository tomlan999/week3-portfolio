About Networks Web Design

Networks Web Design was founded in 2015 by legendary technology visionary, Tom Landeis. Over the course of just a few short months, our business has grown from a desk, chair and laptop in Tom's garage, to a global web development empire.

How does he do it?

Great question! Tom uses the latest tools and services provided by major public cloud service providers to build customer web sites. Key building blocks for customer web sites include:

  • Amazon Web Services provides critical infrastructure and platform services.

  • Microsoft Azure provides key Windows networking and directory services components.

  • Akamai provides content acceleration and delivery, cloud-based security, and protection against DDOS attacks.
